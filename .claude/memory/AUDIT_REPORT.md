## 修复详情

### GAP-001: 引导页弹窗外点击不应关闭
- **状态**: 原设计已满足需求
- **说明**: OnboardingPage 是全屏页面，用户无法通过点击外部关闭，已满足"强制完成引导"的核心需求

### GAP-002: 跨项目选文献功能

- **文件**: `src/pages/ProjectHubPage.tsx`
- **功能**: 用户可从其他项目选择已提取文献加入当前问答上下文

- **文件**: `src/pages/ProjectHubPage.tsx`

- **文件**: `src/pages/ComparisonPage.tsx`

### GAP-005: 收集箱快捷按钮
